## 🏗️ 아키텍처 및 디자인패턴
> MVVM
- MVVM을 도입하여 View가 뷰를 그리는 역할에만 집중하고, 데이터 관리, 로직 실행들을 ViewModel에서 진행되도록 했습니다.

> FactoryPattern
- 팩토리 패턴을 활용하여 코드 실행 시점에 따라 필요한 뷰의 종류를 결정하고, 그에 맞는 뷰 인스턴스를 생성할 수 있도록 했습니다.
